## 🛠️ Getting Started

### Prerequisites
- Node.js 18+ 
- pnpm (recommended) or npm

### Installation

1. Clone repository
```bash
git clone https://github.com/aufadzakiy/web-portoV2.git
cd web-portoV2
```

2. Install dependencies
```bash
pnpm install
# or
npm install
```

3. Run development server

3. Run development server

```bash
pnpm dev
# or
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000)

### Build for Production

```bash
pnpm build
pnpm start
```
